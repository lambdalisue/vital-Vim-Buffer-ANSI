Introductions
-------------------------------------------------------------------------------

To visualize ANSI color on Vim buffer.

Install
-------------------------------------------------------------------------------

Install this repository with your favorite Vim plugin manager and then

```vim
:Vitalize . +Vim.Buffer.ANSI
```

To activate Vim.Buffer.ANSI on your plugin.
